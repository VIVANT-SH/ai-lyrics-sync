# AI 기반 일본어 노래 가사 싱크 서비스

## 프로젝트 목적
일본어 노래의 가사를 음원 재생 시간과 자동으로 매칭하여
학습자가 자연스럽게 일본어 가사를 이해할 수 있도록 돕는 서비스이다.

## 핵심 기능
- 일본어 노래 가사 관리
- 시간대별 가사 싱크 제공
- 일본어 학습 보조 기능 확장 가능

## 기술 스택
- Spring Boot: REST API 서버 구현
- React: 사용자 인터페이스
- PostgreSQL: 가사 및 메타데이터 저장
- Redis: 캐싱 및 성능 최적화

## 이 스택을 사용하는 이유
Spring Boot와 React 조합은 확장성과 유지보수성이 뛰어나며,
PostgreSQL은 텍스트 중심 데이터 처리에 적합하기 때문이다.
